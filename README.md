# ESP32-S3 SubModules

This repository contains simple test programs and examples for exploring the features of the ESP32-S3 Development Kit.

## Purpose

The goal of this repository is to verify and understand the different peripherals and functionalities available on the ESP32-S3 by creating small, independent modules.

## Modules

Some of the components and features tested include:

- GPIO (Digital Input/Output)
- PWM
- ADC
- RGB LED
- I2C Devices
- SPI Devices
- UART Communication
- Sensors
- Interrupts
- Timers
- Other ESP32-S3 peripherals

Each module is kept simple and focused on testing a single feature, making it easy to understand and reuse in future projects.
